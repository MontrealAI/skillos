What to upload

Open this folder and upload its contents to the GitHub repository root:

UPLOAD_THE_CONTENTS_OF_THIS_FOLDER_TO_GITHUB

Do not upload the ZIP file itself to GitHub. Unzip first.

Fastest path

  1. Create MontrealAI/skillos on GitHub.
  2. Click Add file → Upload files.
  3. Drag everything inside UPLOAD_THE_CONTENTS_OF_THIS_FOLDER_TO_GITHUB.
  4. Commit the files.
  5. Go to Settings → Pages.
  6. Set Source to GitHub Actions.
  7. Open Actions and wait for Deploy SkillOS website to GitHub Pages to turn green.
  8. Visit https://montrealai.github.io/skillos/.
