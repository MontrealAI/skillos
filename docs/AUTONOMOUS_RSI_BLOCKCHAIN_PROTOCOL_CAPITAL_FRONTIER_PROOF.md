# Autonomous RSI Blockchain Protocol Capital Frontier Proof

This repository-ready proof is designed to be run by GitHub Actions with no human review.

It tests the blockchain-domain version of the SkillOS thesis:

> Can a large autonomous specialist-agent organization recursively improve how it coordinates capital, blockspace, validator security, MEV control, liquidity, bridges, oracles, governance, data availability, compute/energy, trust, settlement, validation, risk courts, and reinvestment into compounding protocol capability?

## What it generates

The workflow creates:

- `data/rsi-blockchain-protocol-capital-frontier-proof.json`
- `docs/rsi-blockchain-protocol-capital-frontier-proof.md`
- `site/rsi-blockchain-protocol-capital-frontier-proof.html`
- `badges/rsi-blockchain-protocol-capital-frontier-proof.svg`
- `site/index.html`
- `site/proof-registry.json`
- `site/sitemap.xml`
- `site/robots.txt`

## Why it is credible

The proof includes:

- deterministic benchmark generation;
- pre-registered pass/fail gates;
- train / validation / locked holdout separation;
- validation-gated RSI releases;
- single-protocol-strategist baseline;
- uncoordinated-agent-swarm baseline;
- static DAO committee baseline;
- no-RSI protocol organization baseline;
- shuffled-reward, random-protocol, and risk-blind negative controls;
- bootstrap confidence intervals;
- protocol fingerprint;
- generated JSON receipt, Markdown report, badge, proof webpage, and homepage update.

## Safe public boundary

This is a public deterministic benchmark proof. It is not live protocol revenue, audited ROI, investment advice, financial advice, trading advice, a token recommendation, achieved superintelligence, or Kardashev Type II achievement.

The purpose is to make the mechanism testable: a validation-gated, recursively improving, large multi-agent protocol organization coordinating scarce resources into compounding capability.
