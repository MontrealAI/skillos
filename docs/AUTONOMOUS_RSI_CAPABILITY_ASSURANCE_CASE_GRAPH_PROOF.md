# Autonomous RSI Capability Assurance Case Graph Proof

This proof tests whether verified skills can become audit-ready assurance cases.

## Mechanism

```text
capability claim
→ evidence packet
→ control coverage
→ verifier independence
→ red-team challenge
→ residual-risk disclosure
→ release gate
→ public assurance receipt
```

## What it tests

- SkillOS RSI assurance graph vs static checklist
- SkillOS RSI assurance graph vs local audit silos
- SkillOS RSI assurance graph vs paper compliance theater
- SkillOS RSI assurance graph vs risk-blind release
- beautiful machine-generated `Skills Used` cards on the public webpage
- evidence quality, control coverage, verifier independence, audit readiness, assurance gaps, and risk disclosure

## Boundary

Benchmark-capital-equivalent values are not live revenue, customer results, financial guarantees, legal advice, audit certification, policy advice, token advice, medical advice, or proof of achieved superintelligence.
