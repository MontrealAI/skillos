# Autonomous RSI Capability Economy Clearinghouse Proof

This proof tests whether SkillOS can clear a capability economy.

## Mechanism

```text
demand
→ verified skill supply
→ price discovery
→ liquidity
→ settlement trust
→ routing upgrade
→ reinvestment
→ compounding capability economy
```

## What it tests

- SkillOS RSI clearinghouse vs static price book
- SkillOS RSI clearinghouse vs local silo markets
- SkillOS RSI clearinghouse vs subsidy market
- SkillOS RSI clearinghouse vs unverified clearing
- price discovery, settlement trust, verified quality, liquidity, and reinvestment under risk constraints

## Boundary

Benchmark-capital-equivalent values are not live revenue, customer results, financial guarantees, legal advice, policy advice, token advice, or proof of achieved superintelligence.
