# Autonomous RSI Open Replication Mesh Proof

This proof is designed to be run entirely from GitHub Actions.

It tests whether SkillOS public proof receipts survive open replication: independent replica cells replay locked holdout cases, emit signed receipts, converge under verifier courts, and publish a deterministic receipt tree.

Public-safe boundary: this is a deterministic benchmark proof, not live revenue, customer outcomes, financial guarantees, legal advice, investment advice, policy advice, token advice, or proof of achieved superintelligence.

Run:

```text
GitHub → Actions → Autonomous RSI Open Replication Mesh Proof → Run workflow
```

Recommended inputs:

```text
publish_to_repo: true
deploy_pages: false
```

Expected page:

```text
https://montrealai.github.io/skillos/rsi-open-replication-mesh-proof.html
```
