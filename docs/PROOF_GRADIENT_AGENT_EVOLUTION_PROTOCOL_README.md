# Proof Gradient · The Agent Evolution Protocol

One agent tries. Proof decides. The network evolves.

GoalOS gives the network Direction. PlanOS gives it Strategy. SkillOS gives it Capability. Proof Gradient gives it Evolution.

## What this pack adds

This pack adds a GitHub-native, autonomous proof for Proof Gradient. It runs a deterministic benchmark, generates a JSON receipt, renders a beautiful proof page, displays the Skills Used, updates the Public SkillOS Command Center root, deploys GitHub Pages, and verifies the live site.

## Permanent root contract

```text
/skillos/                                            = Public SkillOS Command Center
/skillos/index.html                                  = Public SkillOS Command Center
/skillos/proof-gradient-agent-evolution-protocol.html = Proof Gradient flagship proof room
```

The root is the lobby. Proofs are rooms. A proof room must never replace the lobby.

## What the proof tests

The proof compares a proof-gated evolution protocol against four controls:

```text
single agent
uncoordinated pool
static coordination
unverified propagation
Proof Gradient
```

The proof passes only if Proof Gradient improves holdout performance, increases benchmark value capture, beats unverified propagation, respects a risk ceiling, rejects negative controls, and shows a compounding release curve.

## How to run

Open GitHub Actions and run:

```text
Proof Gradient Agent Evolution Protocol Proof
```

Recommended inputs:

```text
deploy_pages: true
verify_live: true
cancel_legacy_runs: true
```

## Public claim boundary

This proof is a deterministic benchmark and public mechanism demonstration. It does not claim achieved superintelligence, audited customer ROI, investment returns, financial advice, legal advice, medical advice, employment advice, credit advice, token advice, or Kardashev Type II civilization. It shows a testable mechanism: proof-gated skill propagation can outperform static or unverified coordination under the stated benchmark assumptions.
