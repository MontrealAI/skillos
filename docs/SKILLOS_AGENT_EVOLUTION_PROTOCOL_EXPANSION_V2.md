# SkillOS Agent Evolution Protocol Expansion v2

This expansion adds the canonical proof-carrying intelligence architecture to SkillOS:

```text
Aim. Act. Prove. Evolve.

GoalOS gives the network direction.
PlanOS gives it strategy.
SkillOS gives it capability.
The Proof Gradient gives it evolution.

The Artifact Vault stores reusable intelligence.
The Run Fabric executes agents at scale.
The Proof Ledger records what happened.
The Selection Gate promotes only what proved itself.
```

It is additive by design. It does not remove existing proofs, receipts, pages, reports, badges, or workflows unless a repository operator explicitly chooses to retire something separately.
