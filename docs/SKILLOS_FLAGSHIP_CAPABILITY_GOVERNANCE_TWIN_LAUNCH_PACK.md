This pack creates the launch-grade flagship narrative around **Capability Governance Twin** and deploys it autonomously through GitHub Actions.


## What the workflow does

```text
1. Finds the latest Capability Governance Twin proof receipt when present.
2. Normalizes heterogeneous proof JSON safely.
3. Builds a launch-grade public website.
4. Displays Skills Used beautifully.
5. Verifies the root page, flagship page, skills page, receipt, manifest, registry, badge, and local links.
6. Commits generated outputs.
7. Deploys the exact site artifact to GitHub Pages.
8. Optionally verifies the live root and index page.
```

## Recommended run settings

```text
publish_to_repo: true
deploy_pages: true
verify_live: true
cancel_legacy_runs: true
```
