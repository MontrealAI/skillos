# Autonomous RSI Continual Capability Frontier Proof

This proof tests whether SkillOS can keep improving under distribution shift without catastrophic forgetting.

## Mechanism

```text
telemetry
→ drift detection
→ specialist-agent market clearing
→ replay buffer
→ verifier courts
→ rollback gates
→ release promotion
→ multi-regime holdout
→ reinvestment
→ compounding trustworthy capability
```

## Why it matters

A single static benchmark can look impressive. A real RSI system must preserve old capability while adapting to new regimes. This proof makes that failure mode public and testable.
