# Autonomous RSI Corporate Capability Frontier Proof

This is a 100% autonomous GitHub Action proof for the corporate domain.

## Thesis

Recursive-style labs aim to automate knowledge discovery.

SkillOS tests the corporate analogue:

> Can a large autonomous specialist-agent organization recursively improve its own coordination protocol until capital, compute, energy, data, trust, talent, product, distribution, validation, risk control, and reinvestment compound into superior productive capability?

## Why it matters

The corporate version of RSI is not only an agent getting smarter in isolation. It is a superorganization getting better at allocating scarce resources, routing specialized work, validating actions, controlling risk, and reinvesting verified gains into stronger future capability.

## What runs autonomously

The workflow:

1. generates a deterministic corporate capability-frontier benchmark,
2. evaluates a single corporate generalist baseline,
3. evaluates an uncoordinated multi-agent swarm baseline,
4. evaluates a static multi-agent committee baseline,
5. evaluates a no-RSI large-organization ablation,
6. runs validation-gated recursive self-improvement over coordination protocols,
7. evaluates the final fingerprinted protocol on locked holdout cases,
8. runs negative controls,
9. computes bootstrap confidence intervals,
10. writes a JSON receipt,
11. writes a Markdown report,
12. writes a beautiful visual proof webpage,
13. writes a public badge,
14. updates the SkillOS homepage and proof registry,
15. deploys GitHub Pages.

No human review is required.

## Public boundary

This is a deterministic, public, synthetic/redacted-style benchmark proof. It does not claim audited customer revenue, live customer adoption, investment advice, financial advice, achieved superintelligence, or Kardashev Type II civilization. It makes the corporate RSI mechanism publicly testable.
